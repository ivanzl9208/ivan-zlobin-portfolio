# Иван Злобин — продуктовый дизайнер

Одностраничное портфолио, свёрстанное по макету Figma.

## Локальный запуск

```bash
npm run dev
```

После запуска сайт доступен по адресу `http://localhost:4173`.

## Структура

- `index.html` — разметка страницы;
- `styles.css` — стили и адаптивная вёрстка;
- `script.js` — мобильное меню и интерактивные анимации;
- `public/assets` — локальные изображения.
